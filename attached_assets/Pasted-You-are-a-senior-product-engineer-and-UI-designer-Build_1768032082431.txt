Design style:
- Clean
- Modern
- Minimal

========================

Create a reusable dynamic template for each course.

Each course page must include:

- Who this program is for

- Real-world outcomes
- Learning format (online / hybrid / in-person)

- CTA: Enquire Now / Talk to Advisor

Avoid syllabus-style layouts.
Focus on outcomes, roles, and employability.
